# 📸 FrameFusion

A modern Korean-inspired AI Photobooth built with Next.js, React, TypeScript, and Tailwind CSS.

## ✨ Features

- 📷 Live webcam capture
- ⏱ Countdown timer
- 📸 2 / 4 / 6 / 8 photo strips
- 🎨 Multiple photo filters
- 💜 Korean-inspired themes
- ✍️ Live text editor
- 📥 Download photo strip
- 📱 Responsive design

## 🛠 Tech Stack

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS
- React Webcam
- html-to-image
- Framer Motion

## 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/framefusion.git
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```

Open

```
http://localhost:3000
```
